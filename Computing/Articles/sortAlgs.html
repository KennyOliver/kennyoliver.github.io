TL;DR

What's the definition of "sorting"?

In simple English, sorting means organising items by type or category.


How are items sorted?

Items within a dataset are sorted using algorithms.
An algoroithm is a set of instructions; used to process data.

There's a jungle of sort algorithms!

At GCSE (I'm currently in Yr10),
you only need to know 3 different sort algortithms:

Bubble Sort, Insertion Sort, and Merge Sort

See The Algorithms In Action


Bubble

2 items are compared at a time.
The items only swap places if the 2nd item has a smaller value than the 1st.

Take a look at the algorithm visualiser towards the bottom of the page.
You'll see the items "bubble up" to the end of the list.




Insertion

Beginning with the 2nd item,
it is compared to the previous item's (item 1) value.
If the 2nd item is less, it will be placed before the 1st item.

The 3rd item is compared to the previous item (item 2),
and if the value of the 3rd is less than the 2nd,
the 3rd is placed before the 2nd.

This process repeats with each item in succession,
until every item has been processed.
This is like how you organise a handwritten list alphabetically.

Within the entire dataset, a smaller, sorted dataset grows at the beginning.

HWS: See what I mean


Merge

Merge sort is based upon the "Divide & Conquer" plan.

The entire dataset is repeatedly split until each dataset is a single item.

The algorithm then recombines each dataset in groups,
using the insertion sort algorithm.

The entire dataset eventually unifies back into one.
The end result is a sorted dataset!

Just to sum up...

They all have their pros & cons.

Bubble sort is the simplest to implement,
and faster if the dataset is already partially sorted.
However it becomes slower as the dataset increases.

Insertion sort works best with small datasets.

Merge sort takes more steps than bubble sort.
However, when it comes to large datasets, this is outweighed.
Bubble sort is slower.
Nevertheless, merge sort is more complex to integrate.

To sum up even more, each performs best in its ideal conditions.
And also the fact that easiest is not always the best.
